# Автотесты для формы заказа банковской карты

![Java CI with Gradle](https://github.com/iraczar/web-testing-homework/actions/workflows/gradle.yml/badge.svg)

Проект для домашнего задания по автоматизации тестирования веб-интерфейсов.

## Технологии
- Java 11
- Selenium
- JUnit 5
- Gradle

## Запуск тестов
1. Запустить приложение: java -jar artifacts/app-order.jar
2. Запустить тесты: ./gradlew test
